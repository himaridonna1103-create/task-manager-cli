# タスク管理アプリ

CLIでタスクの追加・一覧・完了・削除ができるシンプルなタスク管理ツールです。
タスクはJSONファイルにローカル保存されるため、アプリを閉じても記録が残ります。

---

## 概要・特徴

- タスクの追加・一覧表示・完了・削除
- JSONファイルへの自動保存（`tasks.json`）
- 入力バリデーション（空タイトル・200文字超を弾く）
- エラーハンドリング（不正な入力でもクラッシュしない）
- PEP 8準拠・型アノテーション付き

---

## 動作環境

| 項目 | 要件 |
|------|------|
| Python | 3.8 以上 |
| OS | Windows / macOS / Linux |
| 依存ライブラリ | なし（標準ライブラリのみ） |

---

## インストール

```powershell
# リポジトリをクローン（またはファイルをダウンロード）
git clone <リポジトリURL>
cd <プロジェクトフォルダ>
```

追加インストールは不要です。

---

## 使い方

```powershell
python -X utf8 task_manager.py
```

起動後、以下のコマンドが使えます：

| コマンド | 説明 | 例 |
|---------|------|-----|
| `add <タイトル>` | タスクを追加する | `add 買い物をする` |
| `list` | タスク一覧を表示する | `list` |
| `done <ID>` | タスクを完了にする | `done 1` |
| `delete <ID>` | タスクを削除する | `delete 2` |
| `quit` | アプリを終了する | `quit` |

### 実行例

```
=== タスク管理アプリ ===
コマンド: add / list / done / delete / quit

> add 買い物をする
追加しました: 買い物をする

> add 資料を作る
追加しました: 資料を作る

> list
⬜ [1] 買い物をする
⬜ [2] 資料を作る

> done 1
完了しました: 買い物をする

> list
✅ [1] 買い物をする
⬜ [2] 資料を作る

> delete 2
ID 2 を削除しました。

> quit
終了します。
```

---

## 設定

| 項目 | 内容 |
|------|------|
| 保存ファイル | スクリプトと同じフォルダの `tasks.json` |
| 文字コード | UTF-8 |
| タイトル最大文字数 | 200文字 |

> `tasks.json` は `.gitignore` に追加済みのため、Gitで管理されません。
