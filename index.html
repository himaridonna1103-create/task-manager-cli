CLI アプリ

toco.

シンプルなPython製CLIツール。タスクを追加・完了・削除して、
JSONで自動保存。依存ライブラリ不要。

コードを見る README
$ python -X utf8 task_manager.py
=== toco. ===
コマンド: add / list / done / delete / quit

> add 買い物をする
追加しました: 買い物をする
> list
⬜ [1] 買い物をする
> done 1
完了しました: 買い物をする
> list
✅ [1] 買い物をする
特徴

自動保存

JSONファイルに保存。再起動してもデータが残る。

安全設計

エラーハンドリング・入力バリデーション済み。

依存ゼロ

標準ライブラリのみ。pip install 不要。

テスト済み

pytestでAAAパターンのテスト実装済み。

コマンド一覧
add <タイトル>タスクを追加する
list一覧を表示する
done <ID>タスクを完了にする
delete <ID>タスクを削除する
quitアプリを終了する
インストール

追加ライブラリ不要。Python 3.8以上があれば動きます。

python -X utf8 task_manager.py
